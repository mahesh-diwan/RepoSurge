### Priority Issues

#### [P0] Color system contradicts DESIGN.md — amber ≠ terminal green
DESIGN.md says primary #00FF41 on #0A0A0A. Implementation uses amber #FFB000 on #1A1200. The terminal metaphor is undermined.
Fix: Either update DESIGN.md to match amber implementation, or migrate palette to #00FF41/#0A0A0A.

#### [P0] amber-glow CSS class is undefined
Used in error.tsx, not-found.tsx, and repo/[slug]/not-found.tsx. Never defined in globals.css. Error pages look unfinished.
Fix: Define the class or remove it. Use tailwind-only utilities.

#### [P1] Header stats bar violates "no header metrics" rule
DESIGN.md Don'ts explicitly ban header metrics. Three live counters pull eye upward before the leaderboard.
Fix: Remove or replace with a static tagline.

#### [P1] Panel uses side-stripe border + container shadow
border-l and shadow-2xl both violate DESIGN.md rules.
Fix: Remove border-l, replace shadow-2xl with background-tint elevation.

#### [P1] Nav uses backdrop-blur-lg + container shadow
Glassmorphism (banned) + shadow-lg (banned).
Fix: Remove shadow; evaluate whether backdrop-blur-lg is necessary.
